# Encontrando el Bajón

Un juego de aventura callejera para encontrar comida. Lanza el dado del destino, sigue las cuadras que te indique, junta monedas, resuelve desafíos de observación y descubre dónde termina hoy tu aventura.

## Cómo jugar

1. Presiona **Comenzar aventura**
2. Lanza el dado — te dirá si doblar a la izquierda, derecha, o seguir recto
3. Sigue las instrucciones caminando por la calle
4. Completa desafíos de observación y trivia para ganar monedas
5. Busca el **Tesoro del Día** (un objeto aleatorio en la calle)
6. Después de 10 movimientos puedes terminar la aventura y encontrar el bajón

## Mecánicas

- **Dado del destino** — dirección aleatoria cada turno
- **Desafíos de observación** — encuentra objetos en la calle para ganar monedas
- **Trivia y matemáticas** — preguntas cronometradas; si fallas, retrocedes un movimiento
- **Cartas de destino** — eventos positivos, neutros o desafiantes al azar
- **Tienda del Cazador** — gasta monedas en ventajas, skins de dado, y apuestas
- **Sistema de combo** — encadena desafíos exitosos para bonos extra

## Stack

React (single-file component) · Web Audio API · sin dependencias externas de UI
